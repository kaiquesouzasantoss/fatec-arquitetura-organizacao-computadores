## Enunciado

> Ler um número inteiro e imprimir a saída da seguinte forma:
> - CENTENA = x
> - DEZENA = y
> - UNIDADE = z

**Restrição:**

> *Sabe-se que a entrada deve ser um número entre 100 e 999, validar a entrada para que não permita números fora deste intervalo.

**Exemplo:**

| Entrada | Saída |
| --- | --- |
| 517 | CENTENA = 5 |
| | DEZENA = 1 |
| | UNIDADE = 7 |
